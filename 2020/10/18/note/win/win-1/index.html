power shell 禁止执行脚本的解决方法

2020-10-18

原因

Windows Power Shell执行脚本有一定的策略

  • Restricted 不允许任何脚本执行
  • AllSigned RemoteSigned 防止执行未签名脚本

解决

  1. 使用

    获取脚本执行策略，一般情况为restricted

  2. 使用

    将策略设置为远程脚本需要签名，需要管理员权限
