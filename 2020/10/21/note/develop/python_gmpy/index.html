gmp

gmp是gnu的大整数运算库，gmpy2应用于python平台

安装

  1. 安装python

  2. 安装wheel

  3. 安装gmpy对应的wheel

    在这个网站找到你的python版本对应的gmpy2 wheel，比如说我是python3.8版本就选择gmpy2-2.0.8-cp38-cp38-win_amd64.whl下载之后安装。

    python版本直接在命令行内运行

  4. 安装gmpy2
